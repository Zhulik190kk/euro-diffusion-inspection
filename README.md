Опис проекту
Реалізація завдання **Euro Diffusion** відповідно до лабораторної роботи № 1 з дисципліни «Технологія розробки програмного забезпечення» (або аналогічної).

Програма імітує розповсюдження євро-монет по Європі на спрощеній моделі 10×10 сітки. Кожне місто починає з 1 000 000 монет своєї країни. Щодня монети переміщуються до сусідніх міст (1 монета на кожні 1000 монет одного мотиву). Завдання — визначити, за скільки днів у кожній країні завершиться обіг монет усіх країн.
